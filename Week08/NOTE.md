# XOR-异或

异或：相同为0，不同为1。也可用“不进位加法”来解释。

异或操作的一些特点：

x^0=x

x^1s=~x //注意1s=~0

x^(~x) = 1s

x^x=0

c=a^b => a^c=b,b^c=a //交换两个数

a^b^c = a^(b^c) = (a^b)^c //associative

# 指定位置的位运算

1.将x最右边的n位清零：x&（~0<< n）

2.获取x的第n位值（0或者1）：（x>>n）&1

3.获取x的第n位的幂值：x&（1<<n）

4.仅将第n位置为1：x|(1<<n)

5.仅将第n位置为0：x&(~(1<<n))

6.将x最高位至第n位（含）清零：x&((1<<n)-1)
